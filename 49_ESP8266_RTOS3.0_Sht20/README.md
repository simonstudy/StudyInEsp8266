# IIC Sht20 温湿度传感器 例子  

_This example uses the ESP8266 IIC Master to drive SHT20._

这个例程基于 ESP8266_RTOS_SDK master 分支 SPI 驱动 Sht20 温湿度传感器 。

## 如何使用  

### 硬件接线 

* 接线图:  

| Sht20    | ESP8266 |
| ------- | ------- |
| 3V3     | VCC     |
| SCL     | GPIO4  |
| SDA     | GPIO5  |
| GND     | GND     |
